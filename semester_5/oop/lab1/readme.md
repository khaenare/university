

# Лабораторна робота №1

---

## 1. Загальний опис проєкту

Цей проєкт моделює роботу фургона для транспортування кави.
Програма дозволяє:

* додавати каву різних типів;
* використовувати щільну та нещільну упаковку;
* контролювати зайнятий та доступний обʼєм фургона;
* сортувати каву за ціною за літр;
* шукати товари у діапазоні цін;
* зберігати та завантажувати дані у файл `data.txt`.

Проєкт демонструє роботу з класами, наслідуванням, enum, файлами, колекціями та меню користувача.

## 4. Типи упаковки

### **PackageType.java**

| Тип     | Опис     | Додатковий обʼєм |
| ------- | -------- | ---------------- |
| `DENSE` | щільна   | 0.10 л           |
| `LIGHT` | нещільна | 0.30 л           |

Метод:

* `getVolume()` — повертає додатковий обʼєм упаковки.

---

## 5. Фургон

### **LoadedCoffee.java**

Містить:

* обʼєкт Coffee
* PackageType

### **Van.java**

Поля:

* `maxVolume`
* `currentVolume`
* `coffeeList`

Основні методи:

#### `addCoffee(Coffee coffee, PackageType pack)`

Перевіряє:

* обʼєм кави
* обʼєм упаковки
* доступне місце
  Додає товар або повертає `false`.

#### `sortByPricePerKg()`

Сортування за ціною за літр.

#### `searchByPriceRange(min, max)`

Повертає список кави в заданому діапазоні.

#### `printCoffeeList()`

Виводить усі товари.

#### `saveToFile(String filename)`

Зберігає:

* обʼєм фургона
* всі товари у форматі

```
TYPE;NAME;PRICE;VOLUME;PACKAGE
```

#### `loadFromFile(String filename)`

Завантажує дані назад у список.

## 6. Інтерактивне меню

Меню містить наступні опції:

```
1 — Додати каву
2 — Сортувати за ціною за літр
3 — Пошук за ціною
4 — Показати список кави
5 — Вихід
6 — Зберегти в data.txt
7 — Завантажити з data.txt
```

---

## 7. Формат файлу data.txt

```
VAN_VOLUME=5.0
Ground;Ground Coffee;100.0;0.5;DENSE
Instant;Instant Coffee;120.0;0.4;LIGHT
Bean;Bean Coffee;150.0;0.6;DENSE
```

---

## 8. Демонстраційний режим

У демонстраційному режимі програма:

1. Запитує обʼєм фургона
2. Додає три види кави
3. Виводить початковий список
4. Сортує каву за ціною
5. Шукає каву у діапазоні 100–130
6. Моделює спробу перевантаження фургона
